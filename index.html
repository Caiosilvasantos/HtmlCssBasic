Elementos HTML

H1 até H6

Essa tag é bastante utilizada para criar títulos na pagina Html. Começa pelo h1, que é o mais importante, o texto dele é o maior entre os demais, e o mais chamativo. E vai indo em ordem decrescente perdendo a força e tamanho até o h6, que é o menor desse elemento.

Deve ser usado apenas um h1 por pagina, que deve representar o titulo ou assunto principal de toda a pagina. E deve evitar pular os níveis de titulo, tipo, se usar o h1, o próximo deve ser h2 e por e assim por diante.

Elemento Audio

A tag áudio é usada para implementar um arquivo de áudio no Html5. Ela é usada em conjunto com a tag source, que permite que a tag áudio utilize o arquivo de áudio, que pode ser interpretado pelo navegador.

A grande parte dos navegadores suportam áudios no formato: MP3, WAV, OGG.
